Setup YouTube API

Untuk menggunakan aplikasi ini, Anda perlu API key YouTube

YouTube API Key

Cara mendapatkan API Key:

  1. Buka Google Cloud Console
  2. Buat project baru atau pilih project yang ada
  3. Aktifkan "YouTube Data API v3"
  4. Buat credentials > API Key
  5. Copy dan paste API key di atas

Demo Mode

Jika Anda memilih "Lewati", aplikasi akan berjalan dengan data dummy untuk demo. Anda bisa mengatur API key nanti di menu pengaturan.
